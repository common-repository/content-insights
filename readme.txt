This plugin adds smartocto inisghts (former Content Insights) tracking script to your WordPress blog.

== Description ==

smartocto insights (fomer Content Insights) is a premium content performance analytics tool, delivering unprecedented insights into user behaviour and engagement. 

By signing up for smartocto Insights you're not only getting an unrivalled analytics tool, but also access to world renowned brands looking for content creators to work with. 

After installing the plugin, your article data will be collected by tracking code and insights will be available through the [smartocto insights](https://app.contentinsights.com) web application.

**NOTE:** If your site uses AJAX to load content dynamically (e.g. infinite scroll), please contact our support by sending an email to [support@smartocto.com](mailto:support@smartocto.com) before activating this plugin.

== Installation ==

1. Sign up for smartocto insights (provided by our support team)
1. Download the plugin
1. Upload 'contentinsights' folder to the /wp-content/plugins/ directory
1. Activate the plugin through the 'Plugins' menu in WordPress
1. Under Settings > Content Insights, enter your Site ID manually or try to obtain it automatically
1. Save changes
